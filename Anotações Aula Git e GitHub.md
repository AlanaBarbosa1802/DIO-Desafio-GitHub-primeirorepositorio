Por quê o Git foi criado

Git;É um sofware pronto para nos ajudar, é um sistema de versionamento de arquivos ditribuidos. O Linus precisava ter um sistema que versionace. Ou seja criace versões do trabalho execultado e que fosse complexo o suficiente para que pessoas do mundo inteiro possam trabalhar no mesmo arquivo e e eles ficarem salvos em todas as suas extenções.
É um controle do que já escrevemos, sem que este controle se perca.
**Várias pessoas podem usa-lo ao mesmo tempo**



GitHub; é uma plataforma para ospedar os arquivos 
-Trabalha com repositórios(são as pastas do Git para eu armazenar os meus projetos)
-Pode ser utilizado como portifólio 

Nomes úteis;
Branch-Alternativa dentro do Github para ir salvando o arquivo sem ficar na linha do tempo
marge-junção da Branch ao projeto linha do tempo na GitHub
Remote-Faz uma ligação com que o repositório que você usa subir na plataforma Github
pull-puxa o que tem no repositório do Github para a sua máquina
